# Documents

## Overview
The Documents feature allows you to attach files to client records, accounts, or journal entries. Useful for supporting workpapers, bank statements, signed returns, and engagement correspondence.

## Uploading Files
1. Go to **Tools > Documents**
2. Make sure a client is selected in the sidebar
3. Click **Upload Document**
4. Select a file from your computer (PDF, Excel, Word, images, etc.)
5. Optionally link the document to a specific account or journal entry
6. Add a description and click **Upload**

## Linking Documents
Documents can be linked to:
- **Client-level**: general client documents (engagement letters, signed returns)
- **Account-level**: supporting schedules for a specific account
- **Journal Entry-level**: backup for a specific AJE

## Viewing and Downloading
- Documents are listed in the Documents page with name, date uploaded, and file size
- Click a document name or the **Download** icon to download it
- Use the filter to show documents linked to a specific account or entry

## Deleting Documents
- Click the **Delete** (trash) icon next to a document
- You will be asked to confirm before deletion
- Deleted documents cannot be recovered — download first if you need a local copy

## PDF Import Integration
Documents uploaded as PDFs can be sent to the **AI PDF Import** feature (Tools > Import > PDF) which extracts trial balance data automatically. See the CSV/PDF Import article for details.
