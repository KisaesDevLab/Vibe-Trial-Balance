# Multi-Period Comparison

## Overview
The Period Comparison (Multi-Period) page lets you compare two periods side by side with dollar and percentage variance analysis. This is the foundation for Flux Analysis.

## How to Use
1. Select your client in the sidebar
2. Go to **Reports > Period Comparison**
3. In the comparison controls, select the **Base Period** and the **Compare Period**
4. The table loads automatically showing both periods' balances

## Variance Display
For each account, you'll see:
- Base period balance
- Compare period balance
- **$ Variance**: Base minus Compare
- **% Variance**: (Base - Compare) / |Compare| × 100

## Significance Threshold
- Set a significance threshold (dollar amount) in the controls
- Accounts where |$ Variance| exceeds the threshold are highlighted in yellow/orange
- This helps focus attention on material changes

## Category Grouping
Accounts are grouped by category (Assets, Liabilities, Equity, Income, Expense) with subtotals for each group.

## Variance Notes
- Click any account row to add or edit a **variance note**
- Notes explain why the balance changed (e.g., "New equipment purchase in Q3")
- Notes are saved per account per period pair
- Notes appear on the **Flux Analysis PDF**

## Flux Analysis PDF
- Click **Download Flux PDF** to generate a professional Flux Analysis report
- The report includes all accounts, variances, significance highlights, and variance notes
- Suitable for audit files, client deliverables, and manager review

## Common Use Cases
- Year-over-year comparison (FY2024 vs FY2023)
- Quarter-over-quarter comparison
- Budget vs. actual comparison (if actual and budget are separate periods)
